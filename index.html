# Trading Journal (Minimal) This is a minimal static trading journal page with screenshot uploads and numerology inputs. ## GitHub Pages (make it live) Because this environment isn't authenticated to GitHub, I can't create the repo or push for you. Run the steps below on your machine to publish the site: ```bash git init git add . git commit -m "Initial commit" git branch -M main git remote add origin https://github.com//.git git push -u origin main ``` Then in GitHub: 1. Go to **Settings → Pages**. 2. Under **Build and deployment**, select **GitHub Actions**. 3. The workflow at `.github/workflows/pages.yml` will deploy the site automatically. Your site will be available at: ``` https://.github.io// ``` ## Local preview ```bash python -m http.server 8000 ``` Then open `http://127.0.0.1:8000/index.html`.
